Usando linkas externos

Você pode acessar o repositório público do Curso em Vídeo

Para adicionar um link externo basta usar a tag <a href="LINK">

Você também pode acessar o canal do Youtube

Para abrir um link externo em uma nova janela basta usar a tag <a href="LINK" target="_blank" rel="external">
target="_blank" serve para abrir o link em uma nova janela

Usando links internos

Esta é a primeira página do site. Se você quiser pode acessar outra página

Para adicionar um link interno basta usar a tag <a href="Nome do arquivo">

Você também pode acessar a página de notícias

Você também pode acessar a página teste (duas pastas)
