Principais formatações

Negrito / destaque:

- Neste exemplo temos um termo em negrito usando a tag <b> e </b> (não semântica)

- Neste exemplo temos um termo em negrito usando a tag <strong> e </strong> (semântica)

Itátalico / ênfase

- Nesta frase, temos um termo em itálico usando a tag <i> e </i> (não semântica)

- Nesta frase, temos um termo em itálico usando a tag <em> e </em> (semântica)

Sublinhado

- Neste exemplo temos um termo sublinhado usando a tag <u> e </u> (não semântica)

Texto marcado

- Também podemos criar um texto marcado usando a tag <mark> e </mark>

- Também podemos trocar a cor da marcação usando CSS, com a tag <mark style="background-color: pink;"> e </mark>

Texto grande e pequeno

- Estamos criando um texto grande e um texto pequeno usando as tags <big> e </big> e <small> e </small>

Texto deletado

- Podemos inserir um texto excluído, para ser desconsiderado com a tag <del> e </del>

Texto inserido

- Podemos colocar um texto inserido, para ter destaque com a tag <ins> e </ins>

Texto sobrescrito

- Exemplo de texto sobrescrito x19+3. É só usar a tag <sup> e </sup>

Texto subscrito

- Exemplo de texto subscrito H2O. É só usar a tag <sub> e </sub>
